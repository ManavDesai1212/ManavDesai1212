## Projects

### [VPT Pilot Program Evaluation](./vpt-evaluation)
Causal evaluation of a randomized virtual physical therapy pilot using a Difference-in-Differences estimator (N=1,569). Found a **$926/member/year reduction in healthcare costs** (p=0.012) and a **0.42-point satisfaction gain** (p<0.001). Includes event-study analysis and policy recommendations.

`R` `Econometrics` `Causal Inference` `DiD` `Policy Analysis`

---

### [Financial Discovery Engine](./financial-discovery-engine)
Full-stack financial advisor platform built solo from 0 to 1. Covers client onboarding, automated health scoring, lead scoring, PDF report generation, calendar integration, and CRM pipeline management. Designed for AI-augmented advisor workflows.
